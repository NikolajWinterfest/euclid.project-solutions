## О проекте

Реализован лендинг проектной компании Эвклид со следующими особенностями проекта:

-   Адаптивная кроссбраузерная вёрстка, с пользовательским разрешением от десктопной версии до мобильной 320px
-   Swiper, слайдер реализован для фоновых изображений главной страницы с возможностью переключать слайды кнопками клавиатуры ← →
-   Tabs, Переключение изображения и информации о этапах проекта при клике на кнопку «шаг»
-   Accordion, реализован для секции «Вопросы» с плавной анимацией раскрытия текста
-   Для планшетных устройств и смартфонов реализовано Burger menu и кнопка поиска с плавным раскрытием
-   Кастомный чекбокс для «Пользовательского соглашения»
